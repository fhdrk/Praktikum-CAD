\# Praktikum CAD



\## 📌 Deskripsi

Repository ini berisi file SolidWorks untuk tugas Praktikum CAD.



Project ini mencakup pembuatan beberapa part dasar menggunakan SolidWorks.



---



\## 📂 Struktur Folder



\- `parts/` → File part SolidWorks (.SLDPRT)

\- `assembly/` → File assembly (jika ada)

\- `images/` → Screenshot atau preview desain



---



\## 🛠 Software
